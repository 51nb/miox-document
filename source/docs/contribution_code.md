## 开发设置

你需要安装 Node.js 4+ 版本，clone 该仓库后在仓库根目录运行 `npm install`

## 常用 npm scripts

```shell
# 使用 miox 跑 vue SPA
npm run web
# 使用 miox 跑 react SPA
npm run react:dev
# 使用 miox 跑 ssr，基于 vue
npm run ssr
# 使用 karma 跑自动化测试
npm run test:start
# 将 miox 打包单独一个 js
npm run all:build
```
